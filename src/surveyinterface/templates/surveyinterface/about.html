About

The Survey Data Viewer is a web application for visualizing the results of social science survey data developed as part of the iUTAH Modeling and Data Federation (http://data.iutahepscor.org). For the code and additional documentation, see http://github.com/UCHIC/SurveyDataViewer. The code may be adopted for general use following the stipulations in the code repository (https://github.com/UCHIC/SurveyDataViewer/blob/master/LICENSE).
